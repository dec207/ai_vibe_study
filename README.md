# 개인 목표 기반 AI 바이브코딩 실습 커리큘럼

이 저장소는 서로 다른 OS, AI 구독, 에디터, 모델을 쓰는 3명이 각자 다른 개인 목표를 가지고도 같은 AI 협업 루틴을 경험하도록 만든 2시간 커리큘럼 패키지입니다.

## 파일 구성

- `curriculum.md`: 2시간 전체 진행안과 실습 루틴 설계
- `facilitator-guide.md`: 진행자가 모임 전에 준비하고 현장에서 운영할 절차
- `participant-worksheet.md`: 참가자별 목표 분해, 프롬프트, 검증, 회고 기록지
- `execution-practice-sheet.md`: 현장에서 바로 따라가는 단계별 실행 실습지
- `dist/codex-cli-first-practice-print-manual.pdf`: 인쇄 배포용 매뉴얼
- `dist/codex-cli-first-practice-print-manual.docx`: 수정 가능한 인쇄 배포용 원본

## 운영 원칙

- 결과물은 각자 달라도 됩니다.
- 공통으로 맞추는 것은 도구가 아니라 작업 방식입니다.
- 초보자는 완성도보다 문제 정의, 작은 단위 요청, 검증, 버전 저장, 보안 감각을 익힙니다.
- MCP 서버 제작과 서브에이전트 운영은 공통 필수가 아니라 관찰 또는 선택 체험입니다.

## 첫 모임 산출물

각 참가자는 모임이 끝날 때 다음 3가지를 남깁니다.

1. 자기 목표에 맞는 작은 첫 산출물
2. `participant-worksheet.md`를 복사해 작성한 작업 로그
3. 다음에 이어갈 수 있는 요약과 보안 점검 메모
