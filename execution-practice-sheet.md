# 세부 실행 실습지

이 실습지는 2시간 모임에서 참가자가 각자 다른 목표와 도구를 가지고도 같은 흐름으로 작업하도록 돕는 현장용 문서다. 진행자는 각 구간의 시간을 지키고, 참가자는 빈칸을 채우며 자기 작업을 진행한다.

## 사용 방법

- 참가자별로 이 파일을 복사해서 사용한다.
- 모든 칸을 완벽히 채우는 것보다 작업을 작게 만들고 검증하는 것이 우선이다.
- 설치나 계정 문제로 10분 이상 막히면 웹 기반 도구로 전환한다.
- 실제 개인정보, 비밀번호, API 키, 회사 내부 자료는 넣지 않는다.
